4. ORDER BY

ORDER BY

Päringutulemuste sorteerimiseks kasutatakse ORDER BY klauslit. ORDER BY järele tuleks lisada veerunimed, mille järgi sorteeritakse tulemusi, kui ka sorteerimise suund - ASC kasvav, DESC kahanev. Suuna määramata jätmisel sorteeritakse kirjed vaikimisi kasvavalt. Järgneva kahe päringu seast tagastab esimene päring kõik kirjed pealkirja järgi kasvavalt ning teine päring kahanevalt:

SELECT * FROM film ORDER BY title ASC;
SELECT * FROM film ORDER BY title DESC;

Mitme veeru järgi sorteerimine

Võimalik on sorteerida ka mitme veeru järgi, eraldades need komadega. Järgnev päring sorteerib kirjed kasvavas järjekorras esmalt režissööri ning siis pealkirja järgi:

SELECT * FROM film ORDER BY director ASC, title ASC;
